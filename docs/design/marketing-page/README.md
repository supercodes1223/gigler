# Marketing Page Redesign

Design docs for the new Gigler marketing page. **Docs first, then code.** Nothing in
`src/` changes until the docs here are settled.

## Context

The current landing page was built for a very early version of the product and no
longer matches it. The product direction now follows the Assistant spec
(`PROJECTS/assistant/docs/`, treated as the mature version of Gigler): a personal
assistant you text, call, and email like a real person. Gigler is the final name.

Problems with the current page (see [current-page-audit.md](current-page-audit.md)):

- Does not state what the product is, what it does, or why it matters
- No clear hero section, value prop, or CTA
- Far too technical for the target audience
- Out of date with the actual product

## Audience

The everyman — someone who has never heard of a VPS, an LLM, or an "agent."
They text, call, and email people every day. The page must read like it was
written for them.
