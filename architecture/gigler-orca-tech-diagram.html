Identity = sender (phone / From), never destination — one shared number & one domain serve all users, each isolated in its own GigAgent. Existing production runs on AWS (Amplify Gen 2 · Lambda · DynamoDB · S3 · SES); Track 2 hardening evolves the email channel, per-gig isolation, durable execution & reminders onto the Cloudflare stack shown above. Orca is model-agnostic — it routes each step to the best-fit frontier model via a gateway.
